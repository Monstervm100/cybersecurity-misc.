# Packet Sniffer

Captures and inspects network traffic on a local interface.

## Planned Features
- Live packet capture (e.g. via scapy)
- Protocol breakdown (TCP/UDP/ICMP/DNS/HTTP)
- BPF-style filtering
- Summary statistics and pcap export

## Authorization
Only capture traffic on networks you own or are authorized to monitor.
